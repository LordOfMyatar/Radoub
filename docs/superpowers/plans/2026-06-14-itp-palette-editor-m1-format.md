# ITP Palette Editor — Milestone 1: Format Layer Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `Radoub.Formats` able to read deeply nested ITP palette trees without data loss (#2280) and write an `ItpFile` back to bytes with full round-trip fidelity (new `ItpWriter`).

**Architecture:** ITP files are GFF-based. A reader (`ItpReader`) already exists but (a) drops categories nested under another category, and (b) has no writer counterpart. This milestone fixes the model + reader to carry child categories, then adds a writer that mirrors the reader's field conventions exactly. Verified against real on-disk palettes — every node is GFF struct `Type=0`; nodes carry `STRREF`/`NAME`, optional `ID` (category), optional `LIST` (branch or category children), and blueprints carry `RESREF`. Round-trip is the acceptance gate.

**Tech Stack:** C#, .NET 9, xUnit. GFF helpers in `Radoub.Formats.Gff` (`GffReader`, `GffWriter`, `GffFieldBuilder` static `Add*Field` methods, `GffStruct`, `GffList`).

This is Milestone 1 of 3. Milestones 2 (reorg core in `Radoub.UI`) and 3 (UI control + Trebuchet host) are separate plans that depend on this one. This plan produces a working, testable, mergeable format layer on its own.

Spec: `docs/superpowers/specs/2026-06-14-itp-palette-editor-design.md`

## Reference: real ITP structure (ground truth)

Derived from real `itempalcus.itp` files. Every struct is `Type=0`. Root struct `Type=0xFFFFFFFF`, `FileType="ITP "`, `FileVersion="V3.2"`. Root fields: optional `RESTYPE` (ushort, skeleton only), optional `NEXT_USEABLE_ID` (byte, skeleton only), `MAIN` (list).

Node shapes inside any `LIST`:

- Branch: has `LIST`, no `ID`, no `RESREF`. Carries `STRREF` and/or `NAME`. Children may be branches or categories.
- Category: has `ID` (byte). May also have a `LIST` whose children are blueprints AND/OR nested categories/branches (this nested-category case is what #2280 drops).
- Blueprint: has `RESREF` (non-empty string). May have `STRREF`/`NAME`, and creature-only `CR`/`FACTION`.

Common fields on any node: `STRREF` (uint), `NAME` (string), `DELETE_ME` (string), `TYPE` (byte display type).

Field emission order observed: `STRREF`/`NAME` (name fields) first, then `ID` (categories), then `LIST` (children). Blueprints: name fields then `RESREF` then `CR`/`FACTION`.

private static IEnumerable<PaletteCategoryNode> FlattenCategories(List<PaletteNode> nodes)
{
    foreach (var node in nodes)
    {
        if (node is PaletteCategoryNode category)
        {
            yield return category;
            foreach (var child in FlattenCategories(category.Children))
                yield return child;
        }
        else if (node is PaletteBranchNode branch)

            foreach (var child in FlattenCategories(branch.Children))
                yield return child;
        }
    }
}

### Task 4: Implement `ItpWriter`

**Files:**

- [ ] **Step 1: Implement the writer mirroring reader conventions**

Mirror `UtpWriter`'s shape (static class, `Write(ItpFile)` -> `byte[]`, plus `string`/`Stream` overloads). Build the GFF tree: root `Type=0xFFFFFFFF` with `FileType`/`FileVersion` from the model; emit `RESTYPE`/`NEXT_USEABLE_ID` only when present (skeleton palettes); build `MAIN` from `MainNodes`. Each node is `GffStruct { Type = 0 }`. Emit fields in the observed order: name fields (`STRREF`, `NAME`, `DELETE_ME`), then `ID` for categories, then `TYPE`, then children `LIST`. Blueprints emit name fields then `RESREF` then optional `CR`/`FACTION`. A category's `LIST` contains its `Blueprints` followed by its `Children` (or interleaved — match read order by storing children in one ordered list if round-trip ordering tests require it; if ordering within a category is not load-bearing for Aurora, document that and keep blueprints-then-children).

```csharp
using Radoub.Formats.Gff;
using static Radoub.Formats.Gff.GffFieldBuilder;

namespace Radoub.Formats.Itp;

/// <summary>
/// Writes ITP (Palette) files to GFF binary. Mirrors ItpReader field conventions.
/// Reference: BioWare Aurora ITP format, neverwinter.nim. See ItpReader for the
/// inverse mapping.
/// </summary>
public static class ItpWriter
{
    public static void Write(ItpFile itp, string filePath)
        => File.WriteAllBytes(filePath, Write(itp));

    public static void Write(ItpFile itp, Stream stream)
    {
        var buffer = Write(itp);
        stream.Write(buffer, 0, buffer.Length);
    }

    public static byte[] Write(ItpFile itp)
    {
        var gff = new GffFile
        {
            FileType = itp.FileType,
            FileVersion = itp.FileVersion,
            RootStruct = BuildRoot(itp)
        };
        return GffWriter.Write(gff);
    }

    private static GffStruct BuildRoot(ItpFile itp)
    {
        var root = new GffStruct { Type = 0xFFFFFFFF };
        if (itp.ResType is ushort rt) AddWordField(root, "RESTYPE", rt);
        if (itp.NextUseableId is byte nid) AddByteField(root, "NEXT_USEABLE_ID", nid);

        var main = new GffList();
        foreach (var node in itp.MainNodes)
            main.Elements.Add(BuildNode(node));
        AddListField(root, "MAIN", main);
        return root;
    }

    private static GffStruct BuildNode(PaletteNode node)
    {
        var s = new GffStruct { Type = 0 };
        WriteCommon(s, node);

        switch (node)
        {
            case PaletteBlueprintNode bp:
                AddCExoStringField(s, "RESREF", bp.ResRef);
                if (bp.ChallengeRating is float cr) AddFloatField(s, "CR", cr);
                if (!string.IsNullOrEmpty(bp.Faction)) AddCExoStringField(s, "FACTION", bp.Faction!);
                break;

            case PaletteCategoryNode cat:
                AddByteField(s, "ID", cat.Id);
                if (cat.Blueprints.Count > 0 || cat.Children.Count > 0)
                {
                    var list = new GffList();
                    foreach (var bp in cat.Blueprints) list.Elements.Add(BuildNode(bp));
                    foreach (var child in cat.Children) list.Elements.Add(BuildNode(child));
                    AddListField(s, "LIST", list);
                }
                break;

            case PaletteBranchNode br:
                var blist = new GffList();
                foreach (var child in br.Children) blist.Elements.Add(BuildNode(child));
                AddListField(s, "LIST", blist);
                break;
        }
        return s;
    }

    private static void WriteCommon(GffStruct s, PaletteNode node)
    {
        if (node.StrRef is uint sr) AddDwordField(s, "STRREF", sr);
        if (!string.IsNullOrEmpty(node.Name)) AddCExoStringField(s, "NAME", node.Name!);
        if (!string.IsNullOrEmpty(node.DeleteMe)) AddCExoStringField(s, "DELETE_ME", node.DeleteMe!);
        if (node.DisplayType is byte dt) AddByteField(s, "TYPE", dt);
    }
}
```
